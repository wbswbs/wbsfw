# wbsfw - A PHP Framework

A very basic PHP Framework, to set up Web-Projects fast

## Requisites

  - PHP >= 8.0
  - Composer

## Utilities

  - env
  - Smarty
  - Doctrine


## Einbindung composer.json

"repositories": [
    {
         "type": "vcs",
         "url": "git@github.com:wbswbs/wbsfw.git"
     }
],
