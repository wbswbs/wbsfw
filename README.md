# wbsfw - A PHP Framework

A very basic PHP Framework, to set up Web-Projects fast

## Requisites

  - PHP >= 8.0
  - Composer

## Utilities

  - env
  - Smarty
  - Doctrine

## Expected Structure

  - bin
  - documentation
  - public/
  - var/cache/
  - var/data/
  - var/tmp/
  - tests

## Einbindung composer.json

"repositories": [
    {
         "type": "vcs",
         "url": "git@github.com:wbswbs/wbsfw.git"
     }
],
